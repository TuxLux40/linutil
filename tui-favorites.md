# A list of TUIs for system management on Arch

Every package is from the official repos and runs an Raspberry Pi unless otherwise noted.

## Network & Radio

- **[netscanner](https://github.com/Chleba/netscanner)** - Network scanner
- **nmtui** - ncurses network manager (WiFi management)
- **[oryx](https://github.com/pythops/oryx)** - A TUI for sniffing network traffic using eBPF
- **[termshark](https://github.com/gcla/termshark)** - Terminal UI for tshark
- **[impala](https://github.com/pythops/impala)** - TUI for managing wifi (AUR, doesn't work on RPi4 with Arch)

## Performance Monitoring

- **[atop](https://github.com/Atoptool/atop/)** - root level system and process monitor for Linux
- **[bmon](https://github.com/tgraf/bmon)** - A monitoring and debugging tool for networking statistics
- **[btop](https://github.com/aristocratos/btop)** - Resource monitor with extras
- **[glances](https://github.com/nicolargo/glances)** - Glances an Eye on your system. A top/htop alternative
- **[gotop](https://github.com/xxxserxxx/gotop)** - A terminal based graphical activity monitor
- **[gping](https://github.com/orf/gping)** - Ping, but with a graph
- **[kmon](https://github.com/orhun/kmon)** - Linux Kernel Manager and Activity Monitor
- **[nvtop](https://github.com/Syllo/nvtop)** - GPUs process monitoring for AMD, Intel and NVIDIA
- **[rustnet](https://github.com/domcyrus/rustnet)** - A cross-platform network monitoring tool
- **[trippy](https://github.com/fujiapple852/trippy)** - A network diagnostic tool like mtr and more
- **[gtop](https://github.com/aksakalli/gtop)** - System monitoring dashboard for terminal (AUR)
- **[sockttop](https://github.com/jasonwitty/socktop)** - Remote system monitor with a rich TUI, inspired by top/btop (AUR)
- **[kyanos](https://github.com/hengyoush/kyanos)** - Linux network analysis tool based on eBPF (AUR)
- **[corefreq](https://github.com/cyring/CoreFreq)** - CPU monitoring software designed for 64-bits Processors (AUR)

## Database management

- **[rainfrog](https://github.com/achristmascarl/rainfrog)** - A database management TUI for Postgres, MySQL, and SQLite
- **[lazysql](https://github.com/jorgerojas26/lazysql)** - A cross-platform TUI database management tool written in Go (AUR)

## Development & Build Tools

- **[opencode](https://github.com/sst/opencode)** - AI coding agent, built for the terminal
- **[python-textual](https://github.com/willmcgugan/textual)** - TUI framework for Python
- **[crush](https://github.com/charmbracelet/crush)** - The glamourous AI coding agent (AUR)
- **[lazymake](https://github.com/rshelekhov/lazymake)** - Modern TUI for Makefiles with interactive target selection (AUR)

## Docker & Podman

- **[ctop](https://github.com/bcicen/ctop)** - Top-like interface for container metrics
- **[lazydocker](https://github.com/jesseduffield/lazydocker)** - The lazier way to manage everything docker (AUR)
- **[oxker](https://github.com/mrjackwills/oxker)** - A simple tui to view & control docker containers (AUR)
- **[ducker](https://github.com/robertpsoane/ducker)** - A slightly quackers Docker TUI based on k9s (AUR)

## Files & Storage

- **[caligula](https://github.com/ifd3f/caligula)** - A user-friendly, lightweight TUI for imaging disks
- **[diskonaut](https://github.com/imsnif/diskonaut)** - Terminal disk space navigator
- **[fzf](https://github.com/junegunn/fzf)** - A general-purpose command-line fuzzy finder
- **[ranger](https://github.com/ranger/ranger)** - A VIM-inspired file manager for the console
- **[yazi](https://github.com/sxyazi/yazi)** - Blazing fast terminal file manager written in Rust
- **[ncdu](https://dev.yorhel.nl/ncdu)** - disk usage analyzer with an ncurses interface
- **[zoxide](https://github.com/ajeetdsouza/zoxide)** - A smarter cd command with fuzzy finding
- **[gdu](https://github.com/dundee/gdu)** - Fast disk usage analyzer with console interface written in Go (AUR)

## Misc.

- **[bluetui](https://github.com/pythops/bluetui)** - A TUI for managing bluetooth devices
- **[gpg-tui](https://github.com/orhun/gpg-tui)** - A terminal user interface for GnuPG
- **[intelli-shell](https://github.com/lasantosr/intelli-shell)** - Manage command templates/snippets with dynamic completions and AI integration
- **[khal](https://github.com/pimutils/khal)** - A standards based CLI calendar program
- **[termscp](https://github.com/veeso/termscp)** - A TUI file transfer and explorer, with support for SCP/SFTP/FTP/S3
- **[wavemon](https://github.com/uoaerg/wavemon)** - A wireless device monitoring application
- **[glow](https://github.com/charmbracelet/glow)** - A markdown reader designed to showcase the elegance and capabilities of TUI (AUR)
- **[cronboard](https://aur.archlinux.org/packages/cronboard)** - A terminal-based dashboard for managing cron jobs (AUR)
- **[sshm](https://github.com/gu1llaum-3/sshm)** - SSH made easy and fast: browse, connect, and control from your terminal with a modern TUI
- **[gocheat](https://github.com/Achno/gocheat)** - A beautiful TUI cheatsheet for keybindings, hotkeys, gestures and aliases
- **[multranslate](https://github.com/Lifailon/multranslate)** - A TUI for translating text in multiple translators simultaneously
- **[searxngr](https://github.com/scross01/searxngr)** - Web search TUI for SearXNG
- **[distrobox-tui](https://github.com/phanirithvij/distrobox-tui)** - TUI for managing distrobox containers
- **[nemu](https://github.com/nemuTUI/nemu)** - A TUI for QEMU
